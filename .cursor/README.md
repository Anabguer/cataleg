# Guía Cursor — cataleg_molins

Este directorio concentra la documentación que Cursor y los agentes deben usar como referencia estable del proyecto.

## Orden de lectura recomendado

1. **[AGENTS.md](./AGENTS.md)** — Normas de producto, módulos, UI, `catalog_year`, validaciones y checklist.
2. **[docs/README.md](./docs/README.md)** — Índice de la arquitectura del repositorio (rutas, flujos, archivos clave).

Para cambios en mantenimiento, permisos o nuevas pantallas: normas en `AGENTS.md` + detalle técnico en `docs/`.

## Reglas Cursor (`.cursor/rules/`)

- **`reports-generation.mdc`** — Normas de cabecera y maquetación al generar informes (impresión/PDF); se aplica al trabajar en archivos relacionados con `report_run`, selector, CSS de informes, etc.
