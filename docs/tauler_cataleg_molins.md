# 🧭 Tauler – Catàleg Molins  
## Definición funcional y roadmap

## 🎯 Objetivo del Tauler
El **Tauler** debe ser la pantalla principal de contexto del catálogo por año.  
Debe permitir responder rápidamente a:

- ¿Qué año estoy viendo?
- ¿Qué volumen de datos tiene el catálogo?
- ¿Está todo coherente?
- ¿Dónde tengo que ir ahora?

No debe ser un dashboard complejo, sino **útil, claro y operativo**.
